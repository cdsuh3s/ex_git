# Git Exercise

Cloning

(Local)
```
> pwd
> cd parent
> git clone address
> ls
> cd project
> git remote
> git remote get-url origin
```

```
# edit a file
> gvim README.md

# check status
> git status

# stage the modified file
> git add README.md

# commit the changes
> git commit --all

# push the local changes to the remote repository
> git push
```
