# Git Setup

Open `Git Bash`.

## Setting Git username for every repository

1. Set a git username.
    ```
    $ git config --global user.name Terry
    ```
1. Confirm
    ```
    $ git config --global user.name
    ```


1. Set a git username.
    ```
    $ git config user.name Terry
    ```
1. Confirm
   ```
   $ git config user.name
   ```
